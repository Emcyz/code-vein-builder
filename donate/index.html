Code Vein Builder was created by Seth Clydesdale, to help make sharing builds for Code Vein easier, for free. If you found this project helpful and would like to support it, please see below for the different ways that you can contribute.

  • Sponsor me on GitHub.
  • Send me a tip via PayPal.
  • Buy me a coffee via Ko-fi.
  • Buy me a gift from my Amazon Wishlist.

You can also support my work in other ways.

  • Star the project on Github; Stars show appreciation for a repository maintainer's work.
  • Spread the word about Code Vein Builder; let people know about it or share your builds with others!
  • Feel free to contact me via one of the platforms located under "Contact info" to say thanks.

I greatly appreciate your support, as it helps motivate me to continue working on and improving my projects. Thank you!
